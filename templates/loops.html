Learn Loops

For Loop

The for loop is used to iterate over elements of a sequence,
it is often used when you have a piece of code which you want to repeat
"n" number of times. It works like this: " for all elements in a list, do this "

While Loop

When using a while loop, anything between the While and
the colon, there is a value that first is True but will later be False.
As long as the statement is True , the rest of the code will run.
